## 📦 Зависимости

### Требуется
- Python 3.8+
- RTL-SDR V4 устройство
- rtlsdr.dll (из SDR# или standalone)
- WinUSB драйвер (Zadig)

### Опционально
- numpy (для быстрого FFT)
- matplotlib (для графиков)
- sox (для конвертации аудио)
- Dire Wolf (для APRS декодирования)
- WXtoImg (для NOAA APT)
- SDR# (для GUI приёма)

## 🐛 Известные проблемы

### Windows
- [ ] Асинхронный режим иногда работает нестабильно
- [ ] Большие объёмы сэмплов (>1M) могут вызвать memory error
- [ ] Кодировка консоли (используйте `chcp 65001`)

### Общие
- [ ] DFT медленный для больших объёмов данных
- [ ] Нет автоматической коррекции Допплера для спутников
- [ ] APRS декодирование работает не для всех сигналов
